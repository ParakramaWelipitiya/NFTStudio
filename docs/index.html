System Pipeline

Input & Transformation

Raw PNG assets are imported into the JavaFX UI. A custom 2D affine transformation engine allows real-time spatial manipulation (drag, scale, rotate) directly on the canvas before committing the state.

JavaFX Event Listeners

Database Management

Transformed asset coordinates and categorized layer metadata are securely committed to a local database. This enables high-speed, dynamic CRUD operations for thousands of localized assets without relying on temporary memory lists.

SQLite JDBC

The Graphics Engine

The core generation loop queries the database, selecting random traits based on configured parameters. It utilizes advanced Java image processing to perfectly stack transparent pixels into a single, flattened output asset.

java.awt.Graphics2D javax.imageio.ImageIO

Web3 Integration

Operating synchronously alongside the image compiler, the system automatically formats and generates blockchain-standard .json files, ensuring absolute parity between the visual traits and the IPFS-ready metadata.
